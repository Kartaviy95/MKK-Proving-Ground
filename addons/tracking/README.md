# Описание

Локальное слежение за projectile: tracking-камера, HUD, линии траектории и маркеры снарядов на карте. Камера закрывается через настраиваемую CBA-клавишу “Закрыть камеру” (по умолчанию F). Для ammo с `submunitionAmmo` камера переключается на первую созданную submunition. Камера ведет активный projectile до конца его жизни, затем только при подтвержденном пересечении с поверхностью плавно немного отдаляется от точки impact и удерживает обзор. Если projectile исчез без найденного impact, камера завершается без перехода к земле или воде.
