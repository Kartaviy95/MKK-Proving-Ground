# Описание

Локальное слежение за projectile: tracking-камера, HUD, линии траектории и маркеры снарядов на карте. Камера закрывается через настраиваемую CBA-клавишу “Закрыть камеру” (по умолчанию F). Камера ведет projectile до столкновения, затем вычисляет точку на записанной траектории за 50 метров до последней точки полета, останавливается там, плавно поднимается и удерживает обзор еще 4 секунды для оценки взрыва и повреждений. Дистанция остановки настраивается переменной `_preImpactStopDistance` в `fnc_updateProjectileTrack.sqf`.
