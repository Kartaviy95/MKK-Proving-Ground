# Описание

Глобальный спавн техники, водителей/экипажей, тестовых целей и быстрого маркера на карте, реестр созданных объектов, удаление целей и очистка полигона без обязательной загрузки аддона на dedicated server.

Быстрый маркер на карте создает глобальный `mil_dot` marker и smoke shell в выбранной точке. Marker автоматически удаляется через 3 секунды; smoke shell живет по обычным правилам движка.

Спавн техники поддерживает пустую технику, штатный полный экипаж через `createVehicleCrew` и режим “создать с экипажем”, где создается только водитель классом игрока для клиентского режима водитель+стрелок.
