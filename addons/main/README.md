# Описание

Базовый модуль проекта: конфигурация, CBA keybinds, открытие главного UI, включение свободной камеры, телепорт, метка высоты на карте, быстрый маркер на карте, виртуальный/ACE арсенал, копирование classname, удаление и разблокировка объекта под прицелом. Дефолты CBA-клавиш: камера K, телепорт T, высота точки H, быстрый маркер Q.
