# Описание

Главный интерфейс полигона: browser-стартовое меню, отдельные малые окна дополнительных настроек, экран создания техники, режим водитель+стрелок, свободная камера, телепорт, быстрый маркер и тайминги на карте, перевооружение, тестовые цели, создание взрыва, настройки HUD и обработчики кнопок.

Режим водитель+стрелок запускается из кнопки “Создать с экипажем”: UI получает созданную технику, сажает игрока в слот стрелка и оставляет движение штатным Arma 3 input actions, включая “полный газ”. В этом режиме не использовать ручной `setDriveOnPath` или `setVelocityModelSpace`.

## Дополнительные окна настроек

`MKK_PTG_SettingsDisplay` закрывает главное меню на время настройки. Выбор пунктов применяет параметры и обновляет содержимое малого окна без возврата в главное меню; возврат выполняется только кнопкой «Закрыть».
## Инспектор hitpoints

Инспектор в `fnc_toggleHitpointInspector.sqf` пересчитывает ширину панели, высоту карточек и размеры шрифтов от выбранного размера интерфейса в главном PTG-меню. Для Large/Extra Large окно расширяется, карточка цели получает больше места под displayName/classname, а весь стек при необходимости поджимается по высоте safeZone. Настройки инспектора выбирают категории двигатель/корпус/башня/орудие/колеса/гусеницы/бензобак; колеса и гусеницы разворачиваются во все найденные hitpoints группы, включая левые/правые элементы. При смене размера интерфейса или набора категорий уже открытый инспектор пересобирается, а подписи radial-карточек (`Общий урон`, `Башня`, `Пушка` и другие), строки состояния (`Норма`, критические состояния), classname hitpoint (`HitHull`, `HitTurret` и т.п.) и проценты получают отдельную увеличенную область с автоуменьшением шрифта для длинных строк. Все процентные значения используют единый размер шрифта, чтобы цифры в разных карточках были одинаковой высоты.

Свободная камера запускается кнопкой “Камера” в начале центрального столбца dashboard или настраиваемой CBA-клавишей “Камера” (по умолчанию K), показывает скорость при прокрутке колесика, переключает ночное видение по N и закрывается настраиваемой клавишей “Закрыть камеру” (по умолчанию F). Телепорт доступен кнопкой “Телепорт” в начале центрального столбца dashboard или настраиваемой клавишей “Телепорт” (по умолчанию T) и принимает обычный ЛКМ по карте. Стартовый dashboard показывает табло действия кнопок в левой колонке и компактный список важных CBA-клавиш с текущими пользовательскими назначениями в правой колонке.

Быстрый маркер на карте использует web-настройку цвета дыма в dashboard и локальный обработчик обычной карты: настраиваемая CBA-клавиша `Быстрый маркер на карте` (по умолчанию `Q`) ставит маркер под курсором и отправляет запрос в `ptg_spawn` на создание глобального маркера с именем игрока и smoke shell выбранного цвета. Marker удаляется через 3 секунды.

Тайминги на карте находятся в компактном нижнем блоке dashboard рядом с настройкой дыма. Единая кнопка “Старт/Стоп” или настраиваемая CBA-клавиша “Старт/Стоп таймингов” (по умолчанию `Ctrl+X`) запускает или завершает локальный замер движения игрока, сразу ставит стартовый marker `00:00`, а “Очистить тайминги” удаляет локальные timing-маркеры текущего клиента. Шестеренка открывает web-настройки цвета marker, шага постановки меток (по умолчанию 10 секунд) и опционального добавления скорости в подпись marker. Во время активного замера справа сверху под штатной информацией оружия показывается локальный HUD-таймер.

Метки высоты на карте всегда активны и имеют web-настройку цвета в нижнем блоке картографических инструментов dashboard. Сама метка ставится на открытой карте настраиваемой CBA-клавишей “Высота точки” (по умолчанию `H`). Цвет хранится локально в профиле игрока и применяется к новым и уже созданным локальным меткам высоты.
