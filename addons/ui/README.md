# Описание

Главный интерфейс полигона: стартовое меню, отдельные малые окна дополнительных настроек, экран создания техники, режим водитель+стрелок, свободная камера, телепорт, перевооружение, тестовые цели, настройки HUD и обработчики кнопок.

Режим водитель+стрелок запускается из кнопки “Создать с экипажем”: UI получает созданную технику, сажает игрока в слот стрелка и оставляет движение штатным Arma 3 input actions, включая “полный газ”. В этом режиме не использовать ручной `setDriveOnPath` или `setVelocityModelSpace`.

## Дополнительные окна настроек

`MKK_PTG_SettingsDisplay` закрывает главное меню на время настройки. Выбор пунктов применяет параметры и обновляет содержимое малого окна без возврата в главное меню; возврат выполняется только кнопкой «Закрыть».
## Инспектор hitpoints

Инспектор в `fnc_toggleHitpointInspector.sqf` пересчитывает ширину панели, высоту карточек и размеры шрифтов от выбранного размера интерфейса в главном PTG-меню. Для Large/Extra Large окно расширяется, карточка цели получает больше места под displayName/classname, а весь стек при необходимости поджимается по высоте safeZone. При смене размера интерфейса уже открытый инспектор пересобирается, а подписи radial-карточек (`Общий урон`, `Башня`, `Пушка` и другие), строки состояния (`Норма`, критические состояния), classname hitpoint (`HitHull`, `HitTurret` и т.п.) и проценты получают отдельную увеличенную область с автоуменьшением шрифта для длинных строк. Все процентные значения используют единый размер шрифта, чтобы цифры в разных карточках были одинаковой высоты.

Свободная камера показывает скорость при прокрутке колесика, переключает ночное видение по N и закрывается настраиваемой клавишей. Телепорт также доступен через настраиваемую клавишу “Телепорт” (по умолчанию T).
