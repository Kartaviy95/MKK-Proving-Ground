# MKK Proving Ground V1 Dev Kit

Стартовый каркас для нового полигона Arma 3 с префиксом `mkk_ptg`.

## Состав
- `mkk_ptg_core`
- `mkk_ptg_ui`
- `mkk_ptg_catalog`
- `mkk_ptg_spawn`
- `mkk_ptg_targets`
- `mkk_ptg_tracking`
- `mkk_ptg_player`
- `mkk_ptg_ace`
- `mkk_ptg_common`

## Зависимости
- Arma 3
- CBA_A3
- ACE3 (для модуля `mkk_ptg_ace`)

## Что уже есть
- каркас модулей и `config.cpp`;
- регистрация CBA settings и keybind;
- базовый UI-каркас;
- динамический каталог техники;
- серверный спавн техники;
- базовые цели;
- локальный tracking projectile;
- локальные инструменты игрока: бесконечные патроны и режим бога;
- ACE self-action и action на терминале;
- заготовка `CfgRemoteExec`.

## Что нужно проверить в игре
1. Модуль загружается без ошибок.
2. Клавиша открытия интерфейса работает.
3. Каталог собирается и фильтруется.
4. Карточка техники обновляется.
5. Спавн техники работает на dedicated.
6. Мишени создаются и очищаются.
7. Tracking запускается и корректно останавливается.

## Примечание
Это именно стартовый dev-kit для теста и дальнейших правок.
Некоторые части сделаны как безопасный каркас, который можно постепенно расширять.
