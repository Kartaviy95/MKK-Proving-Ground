# MKK Proving Ground

Модульный полигон для Arma 3 с техническим именем `mkk_ptg` и префиксом сборки `ptg`. Текущая сборка: `1.3.3.0`, минимальная версия Arma 3: `2.20`. Проект предназначен для быстрой проверки техники, вооружения, боеприпасов, траекторий, пробития, телепортации и локальных утилит игрока.

## Состав модулей

- `ptg_main` — базовая конфигурация, горячие клавиши, открытие UI, виртуальный/ACE арсенал, копирование classname, удаление и разблокировка объекта под прицелом.
- `ptg_common` — общие helpers, безопасное чтение конфигов, локализация на English/Russian/German, масштабирование UI/HUD.
- `ptg_catalog` — динамический каталог `CfgVehicles`, фильтрация, сортировка, определение типа техники и совместимых ящиков БК для статики.
- `ptg_ui` — главное окно, стартовое меню, отдельные малые окна дополнительных настроек, экран создания техники, режим водитель+стрелок, камера, телепорт, перевооружение, тестовые цели, настройки HUD, инспектор hitpoints, масштаб интерфейса и обработчики кнопок.
- `ptg_spawn` — глобальный спавн техники, водителей/экипажа и тестовых целей, регистрация созданных объектов, удаление и очистка полигона без обязательной загрузки аддона на dedicated server.
- `ptg_tracking` — слежение за projectile, HUD tracking, линии траектории и маркеры снарядов на карте.
- `ptg_penetration` — тест пробития техники, orbit-камера, тестовый projectile, отчет по damage/AllowDamage/hitpoints/crew и параметры `CfgAmmo`.
- `ptg_player` — бесконечные патроны, режим бога, обработчики выстрелов и респавна.
- `ptg_player_ace` — совместимость режима бога с ACE medical damage.
- `ptg_ace` — ACE self-action и ACE action на терминалах.

## Зависимости

- Arma 3 версии не ниже 2.20.
- CBA_A3.
- ACE3 для модулей `ptg_ace` и `ptg_player_ace`; оба модуля помечены как optional через `skipWhenMissingDependencies` и пропускаются, если ACE не загружен.

## Текущий функционал

- чистое стартовое меню вместо открытия каталога сразу;
- динамический каталог техники из `CfgVehicles` без жесткой привязки к модовым фракциям;
- фильтры по стороне, фракции, типу техники и поиску по названию/classname;
- карточка выбранной техники с изображением, фракцией, типом, группой, экипажем и источником мода;
- глобальный спавн пустой техники, техники с полным экипажем или техники с водителем класса игрока и игроком в слоте стрелка;
- управление техникой из слота стрелка через штатные Arma 3 actions движения, включая “полный газ”;
- совместимые ящики БК для статического вооружения;
- отдельное меню тестовых целей и глобальное создание/удаление целей;
- телепорт по клику на карте или через настраиваемую клавишу “Телепорт” (по умолчанию T);
- свободная камера: выбор области на карте, создание камеры в выбранной точке, управление WASD/Q/Z/Shift/мышью/колесом, временная индикация скорости при прокрутке, ночное видение по N, подсказка управления по F1, перенос камеры по ЛКМ и закрытие через настраиваемую клавишу;
- перевооружение техники игрока: выбор позиции экипажа/турели, выбор оружия, загрузка совместимых магазинов, очистка оружия и работа с пилонами через `setPylonLoadout`;
- tracking projectile с локальной камерой, HUD по ammo/времени полета/дистанции/режиму/скорости, таймаутом и cooldown;
- линии траектории с настройкой цвета и толщины;
- маркеры снарядов на карте с опциональной подписью ammo classname;
- статус-дисплей объектов при наведении и настраиваемые поля damage/AllowDamage/hitpoints;
- инспектор hitpoints для техники под прицелом с компактными radial-карточками общего урона и ключевых hitpoints;
- тест пробития с выбором техники и боеприпаса, orbit-камерой, выстрелом по выбранной точке, отчетом повреждений и копированием ammo classname;
- бесконечные патроны и режим бога с повторным применением после респавна;
- виртуальный арсенал и ACE3 Arsenal через горячие клавиши;
- копирование classname объекта под прицелом;
- удаление объекта под прицелом с проверкой доступа и защитой от удаления игроков;
- разблокировка техники под прицелом;
- масштаб интерфейса: small, normal, large, extra large;
- локализация видимого текста через `addons/main/stringtable.xml` на английский, русский и немецкий языки; в текущей stringtable 300 ключей, все имеют English, Russian и German.

## Документация

Основные актуальные документы находятся в `docs/ptg/`:

- `overview.md` — идея продукта, стиль и принципы;
- `ui-ux.md` — структура интерфейса, стартовое меню, экран техники, камера, перевооружение и оверлеи;
- `catalog-and-spawn.md` — каталог, фильтры, спавн, ящики БК и очистка;
- `features.md` — телепорт, камера, tracking, траектории, маркеры, hitpoints, тест пробития, доступ и настройки;
- `architecture-and-localization.md` — модули, зависимости, UI-поверхности, игровые потоки и правила локализации;
- `design-full.md` — полный исходный дизайн, использовать только если раздельных документов недостаточно.

README модулей в `addons/*/README.md` являются краткими заметками; при расхождениях приоритет имеют `docs/ptg/*.md` и текущий код.
